Â¿COMO EJECUTAR EL PROYECTO? SIGA ESTOS PASOS:

1. Descargue el proyecto, y agreguelo a su IDE (Visual Studio Code preferentemente)

4. Una vez descargadas, escriba el comando "npm start".
